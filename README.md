## MINI PROJECT CYPRESS AUTOMATION TEST - PERSONAL PORTFOLIO WEBSITE 

## 📋 Project Overview
Project ini dibuat sebagai latihan QA Automation menggunakan **Cypress**, untuk menguji fungsionalitas, responsivitas, dan tampilan visual dari website portfolio pribadi.
Project ini merupakan inisiatif pribadi untuk memperdalam pemahaman **end-to-end testing** dengan skenario nyata.

## 🧰 Tech Stack
- **Cypress 13+**
- **JavaScript (ES6)**
- **Node.js**
- **Vercel Hosting (Website under test)**

## 🌐 Website Under Test
🔗 [https://azzahra-portfolio-site.vercel.app/](https://azzahra-portfolio-site.vercel.app/)

## 🧩 Test Categories & Coverage

### 🧠 Functional Testing
- Navbar dan Footer Menu Navigation
- Button Functionality Check
- Form Validation dan Submission (dengan reCAPTCHA simulation)  
- Social Media & External Links  
- Hamburger Menu (Responsive Navigation)

### 📱 Responsive Testing
- Viewport test (Desktop, Tablet, Mobile)  
- Layout visibility (header, main, footer)  
- Overflow / horizontal scroll check  

### 🎨 Visual Testing
- Image load verification (no broken image)
